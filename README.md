# CaCC-Lab 🔧  

**システムエンジニア × 臨床心理士 × ICT 教育支援 × キャリアコンサルタント**  

---

## 🎯 ミッション  
- 技術と人間理解の両輪で、「人に寄り添う ICT 活用」と「若者のキャリア支援」を実践する  
- ソフトウェア開発経験と心理的知見を活かし、**“人に優しい設計と支援”** を形にする  

---

## ⚙️ スキル & 経験  

| 分野 | 主な内容 |
|------|----------|
| 💻 ソフトウェア開発 | Java Servlet / JSP を用いた業務系開発（設計・運用含む） — 約 7.5 年 |
| 🧠 心理／キャリア支援 | 臨床心理学修士／公認心理師への進路、若者の就労支援、小学校での ICT 支援経験 |
| 📚 教育・ICT 支援 | 小学校 ICT 支援員、教材開発、プログラミング支援、タブレット／教材設計 |
