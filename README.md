## Development Philosophy

### Core Principles
1. **Security First** - パッチ即時適用、OWASP Top 10 準拠
2. **TDD (Test-Driven Development)** - テスト → 実装 → リファクタリング
3. **5AI Collaboration** - 複数 AI の強みを活かした協調開発
4. **Human-Centered Design** - 心理学知見を活かした UX 設計
5. **Documentation as Code** - コードと同等の品質でドキュメント管理

### Quality Standards
- **Testing Trophy**: Unit (20%) → Integration (60%) → E2E (20%)
- **Error Handling**: `{ what, why, how }` パターン
- **Logging**: VibeLogger による AI-native 構造化ログ
- **Security**: JWT, CSRF 対策, XSS 対策, SQLi 対策
